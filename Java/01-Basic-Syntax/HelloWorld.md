# Hello World

## 程序结构

```java
package day01;

public class HelloWorld {

    public static void main(String[] args) {

        System.out.println("hello world");

    }

}
```

---

## 知识点

### package

用于声明当前类所在的包。

可以理解为文件夹。

---

### class

Java 的基本组成单位。

类名必须与文件名一致。

---

### main()

Java 程序入口。

程序运行时首先执行：

```java
public static void main(String[] args)
```

---

### System.out.println()

控制台输出。

例如：

```java
System.out.println("Hello");
System.out.println(100);
```

---

## 注意

- 每个 Java 程序都有入口。
- main 方法固定写法需要记住。
